# SkillFactory QAP-89

Тестирование бота предоставленного сокурсником в рамках практического задания 18.6.2

В ходе тестирования были выявлены следующие ошибки:

1. В файле config.py в словаре keys неверно указано значение для валюты 'доллар'.
    ожидаемо: 'USD'
    фактически: 'ЮэСДэ'

2. Бот принимает в качестве количества только целые значения.
    ожидаемо: Можно указать дробное количество валюты для конвертации (например "доллар рубль 5.5")
    фактически: При указании дробного количества выводится ошибка
