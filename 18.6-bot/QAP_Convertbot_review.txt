# SkillFactory QAP-89

Тестирование бота предоставленного сокурсником в рамках практического задания 18.6.2

В ходе тестирования были выявлены следующие ошибки:

1. В файле config.py в словаре keys неверно указано значение для валюты 'доллар'.
    ожидаемо: 'USD'
    фактически: 'ЮэСДэ'

2. Бот принимает в качестве количества только целые значения.
    ожидаемо: Можно указать дробное количество валюты для конвертации (например "доллар рубль 5.5")
    фактически: При указании дробного количества выводится ошибка "Не смог обработать количество 5.5"

3. Бот при конвертации не учитывает количество.
    ожидаемо: Ответ бота должен учитывать количество валюты указанное пользователем
    фактически: Независимо от количества валюты указанного пользователем бот всегда выдает курс обмена одной еденицы валюты

4. Бот при расчёте конвертации путает валюты между собой. Для исправления можно в файле extensions.py поправить строки 33 и 34
    ожидаемо: f'https://min-api.cryptocompare.com/data/price?fsym={quote_ticker}&tsyms={base_ticker}')
              total_base = float(json.loads(r.content)[keys[base]])
    фактически: 33 f'https://min-api.cryptocompare.com/data/price?fsym={base_ticker}&tsyms={quote_ticker}')
                total_base = float(json.loads(r.content)[keys[quote]])

5. В файле app.py в строке 49 бот запускается без параметра "none_stop=True"
    ожидаемо: bot.polling(none_stop=True)
    фактически: bot.polling()

6. Все файлы заканчиваются строкой с данными, а не пустой строкой.
